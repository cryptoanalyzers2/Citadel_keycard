Scope

With the development of new JCOP releases, new JavaCard API Extensions for JCOP products have been introduced. To ensure that you are able to load your developed JavaCard applet onto a simulation or a JCOP card, you need to check which version of JavaCard API Extension is available on your target.
If there is a version mismatch, the upload process will abort with an error.
